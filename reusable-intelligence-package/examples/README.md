# Reusable Intelligence Package Examples

This directory contains practical examples demonstrating how to use the reusable components in the package.

## Available Examples

### 1. MCP Tool Generator
- `mcp_tool_example.py` - Shows how to generate an MCP tool for a custom use case

### 2. Agent Integration
- `agent_integration_example.py` - Demonstrates integrating sub-agents into a new application

### 3. Prompt Generation
- `prompt_generation_example.py` - Shows how to use the prompt generator for different agent types

### 4. Complete Implementation
- `full_implementation_example.py` - Complete example of using multiple components together

## Getting Started

To run the examples:

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run an example:
   ```bash
   python mcp_tool_example.py
   ```

Each example includes detailed comments explaining the implementation and customization options.
