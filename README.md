<h1 align="center">AI Chat Bot</h1>
<p align="center"><strong>Intelligent conversational assistant powered by OpenAI API with context-aware responses.</strong></p>

---

## Features

- **Context-Aware Conversations** — Maintains conversation history for coherent multi-turn dialogue
- **OpenAI Integration** — Powered by GPT models via OpenAI API
- **Real-Time Responses** — Streaming support for instant feedback
- **Clean Interface** — Minimal, user-friendly design
- **Extensible** — Easy to add custom tools, personas, or knowledge bases

---

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Backend | Python |
| AI Model | OpenAI GPT |
| API | OpenAI SDK |

## Quick Start

```bash
git clone https://github.com/Ub207/ai-chat-bot.git
cd ai-chat-bot
pip install -r requirements.txt
export OPENAI_API_KEY=your_key_here
python app.py
```
